## :star: Stack

-   [TypeScript](https://www.typescriptlang.org) | A typed superset of JavaScript that compiles to plain JavaScript
-   [React](https://reactjs.org) | A JavaScript library for building user interfaces
-   [Stitches](https://stitches.dev) | A fully-featured styling library (CSS-in-JS)

## :fire: Getting started

```bash
# Clone the project
$ git clone git@github.com:kayooliveira/kayooliveira.com.git

# Access the project folder
$ cd kayooliveira.com

# Install dependencies (you can use npm or another package manager as well)
$ yarn

#Run the development server
$ yarn dev
```

> The app will be available on your browser at the following url `http://localhost:3000`
